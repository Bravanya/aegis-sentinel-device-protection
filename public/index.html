Windows device.

          
● AGENT OFFLINE

        

        
IMPORTANT:This page communicates only with an optional local agent running on 127.0.0.1 on your own PC. It can check Microsoft Defender, launch a Defender scan, and block a specific IP with Windows Firewall after you authorize it. It cannot scan, control, or access another person's laptop, Google account, LinkedIn account, or network.

        

          

            
1 · CONNECT THIS PC
Local security agent
 NOT CONNECTED

            

              
Start agent\aegis-device-agent.js on this Windows computer. The agent generates a private token in agent-config.json. Paste it here for this browser session only.

              
              

              
To enable firewall changes:run start-agent-admin.ps1 and approve the Windows UAC prompt.

These buttons ask your local agent to launch Windows Defender on this computer. Scan results remain in Windows Security.

              

              
Quick scan:checks common malware locations. Full scan: may take a long time depending on your disk size.
